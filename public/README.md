# Public Assets Folder

Place your panda icon here to replace the emoji on the landing page.

## Panda Icon

**File name:** `panda-icon.png` (or `.svg`, `.jpg`, `.webp`)

**Location:** `/public/panda-icon.png`

**Recommended size:** 80x80px to 160x160px (square)

**Supported formats:**
- PNG (recommended for transparency)
- SVG (scalable, best quality)
- JPG
- WEBP

The app will automatically use your icon if it exists, otherwise it will fallback to the üêº emoji.

To change the icon path, edit `src/components/PandaLogo.tsx` and update the `pandaIconPath` variable.
